Main information

This is a simple learning project that demonstrates basic skills in working with several major Java developer frameworks. It is a website where users can publish their own articles of small content, share any interesting information and find one that will be useful to them.

Check my other works

Reflection

Although the project does not stand out for the special beauty of the user interface, it took a lot of time to make it at least look good, which Bootstrap helped a lot (and also created a lot of problems)

Technologies

The following technologies were used in the project:
- Spring Framework: Spring Boot, Spring Core, Spring Data JPA, Spring Security
- Maven
- Hibernate ORM
- PostgreSQL
- Web template engine ‘Thymeleaf’
- Bootstrap
